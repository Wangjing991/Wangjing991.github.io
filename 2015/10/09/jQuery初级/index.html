jQuery初级

  • jQuery

这篇文章介绍jQuery基础知识

$("#div").click(function(){
        alert($(this).html())  //jq写法
        alert(this.innerHTML)  //js写法
    })
    1.jQuery当中忽略等号用传参的方式代替等号
    2.方法函数化需要在方法后面加()
    例:$("ul").children().css('background','red')
    jQuery操作方法函数化
    3.jQuery的写法基本
    var oDiv=$("#div")
    oDiv.html("hello")
    oDiv.css("background","red")
    oDiv.click(function(){
        alert(123)
    })
    4.jQuery的链式写法
    $("#div").html("hello").css("background","red").click(function(){
        alert(123)
    })
    5.取值赋值合体
    $('#div').html("hello")  //赋值
    $('#div').html("")   //取值
    css('width','200px')//赋值
    css('width')//取值
    alert($('li').html()) //当一组元素取值取第一个
    $('li').html('hello') //当赋值的时候赋值所有的li

    6.
    attr 获取属性
    $('div').attr('title','456')

    filter:过滤
    not:filter的反义词
    $('div').filter('.box').css('background','red')
    只有在div中带有.box的才执行

    $('div').has('span').css('background','red')
    只有在div中包含有span标签的才执行
    filter与has区别:filter是编列的
                    has是包含关系

    7.next和prev()
    next下一个兄弟节点
    prev()上一个兄弟节点
    $("div").next('h2').css('background','red'
    例:    <div></div>
        <h2></h2>
    8.find()
    div 下面的h2标签
    $("div").find('h2').css('background','red')
    9.eq()相当于下标的作用

    10.index()一组元素的索引
    当前元素在所有兄弟节点间的位置
    例:<h2></h2>
        <h2></h2>
        <h2></h2>
        <h2></h2>
    11.width和height
        innerWidth
        outerWidth 
        $('div').innerWidth()// width+padding
        $('div').outerWidth()// width+padding+border
        $('div').outerWidth(true)// width+padding+border+margin
        $("div").width()//获取的是不带px的宽度  200；
    12.addClass()添加class属性
       removeClass()移除class属性

    13.insertBefore
        insertAfter
        appendTo
        prependTo
    $('span').insertBefore($('div'))

    14.on() 运用
    $("div").click(function(){

    })
    $('div').on("click mouseover",function(){

    })
    $('div').on({
        'click':function(){
            alert(123)
        },
        'mouseover':function(){
            alert(456)
        }
    })

    15.off()取消事件点击一次就失效
        $('div').on("click mouseover",function(){
        alert(123)
        $('box').off('click')
    })
    16.ev.pageX相对于文档对象的位置
        clientX相对于可视区域
        which:keycode 键盘的键值
        ev.preventDefault阻止默认事件
        ev.stopPropagation阻止冒泡
        return false 阻止默认事件和阻止冒泡
    17.one()相当于事件只执行一次就消失
        $('div').one("click mouseover",function(){
        alert(123)
        $('box').off('click')
    })
    18.Offset获取div2相对于屏幕的距离不管父级有没有定位
    $("#div2").offset().left
    <div id="div1">
        <div id="div2">
        </div>
    </div>
    Position获取到有定位的父级的距离（相当于js中offsetleft)
    position().left
    parent获取父级
    parent  div2父级是div1
    parentsUntil("html","div")//一参：直到么一标签；二参：筛选区间内的div
    offsetParent获取有定位的父级(Position定位)
    Offsetparent    :如果没有定位div2父级就是body
                若有定位div2父级就是div1
    19. val 获取value值
        size 获取元素的长度相当于length
        each相当于for
        一参:下标 二参:函数
