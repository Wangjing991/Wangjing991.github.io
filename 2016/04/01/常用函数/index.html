1.类typeof函数

在Javascript中提供了typeof函数来f可以准确地判断出简单数据类型，但对复杂数据类型的判断相对比较模糊。typeof 返回值有六种可能： “number,” “string,” “boolean,” “object,” “function,” 和 “undefined.

比如： 数组对象 \ 时间对象 \ 正则对象 \ Math对象等都返回的是 ‘object’

优化方案：

封装函数
  function type( obj ){
       var o = {};
       return     o.toString.call(obj).slice(8,-1).toLowerCase(); 
     }

实现原理：
Object 对象的 prototype 的 toString() 方法会返回一个表示对象本身类型的字符串。

2.isArrayLike判断数组类型

在工作中经常遇到判断一个对象是否是数组或者类数组，因此构造函数isArrayLike判断对象是否是数组或者类数组

function isArrayLike( obj ){
       var tp = type(obj),length;
       if(!obj||tp==='function'||tp==='window'||tp==='string'||tp==='object') return false;

       if( tp === 'array' ) return true;

       length = obj.length;

       if('length' in obj && typeof length === 'number' &&  length >= 0 ) return true;
       //判断是否有length属性，并且length是一个大于等于0的数字 

       return false;
     }

     function type(obj){
       var o = {};
       return o.toString.call(obj).slice(8,-1).toLowerCase();
     }

PS： in 运算符 

-  检测对象中是否有指定的属性   
 -   '属性名' in 对象    
 -  有 true   没有  false

3.each循环遍历函数

封装一个函数each（）函数，可以实现遍历数组、js对象及DOM对象。

function each(obj,callback){
        var i = 0, len;
        if(!obj||typeof obj !== 'object')return;
        if( isArrayLike(obj) ){
          len = obj.length;
          for(; i<len; i++){
            callback.call(obj[i],i,obj[i]);
          } 
        }else{
          for(i in obj){
            callback.call(obj[i],i,obj[i]);
          } 
        }
      }
      function isArrayLike( obj ){
        var tp = type(obj),length;
        if(!obj||tp==='function'||tp==='window'||tp==='string'||tp==='object') return false;

        length = obj.length;
        if('length' in obj && typeof length === 'number' &&  length >= 0 ) return true;
        return false;

4.获取DOM元素属性函数

function css(obj,attr){
  return parseFloat( obj.currentStyle ? obj.currentStyle[attr] : getComputedStyle(obj)[attr] ) || 0;
}

5.线性运动函数

function doMove(obj,attr,target){
  var iLeft = css(obj,attr);
  var timer = null;
  timer = setInterval(function(){
    iLeft += 10; 
    if( iLeft >= target ){
      iLeft = target;
      clearInterval(timer);
    }   
    obj.style[attr] = iLeft + 'px';  
  },20);  
}
