本地文件上传到github

  • github

上传的步骤：



1、先建立一个个人仓库，作为容器来装文件。打开个人的github主页，New repository （注意：命名的仓库文件夹最好不要用中文）。

2、建好仓库容器之后，git clone https://github.com/kangyuf/ceshibaoxiang.git 在桌面找到刚建立的文件，把要上传的文件拉进去。

3、在ceshibaoxiang下git add .（.的含义是包含所有文件）

4、然后 git status（查看上传文件的状态）

5、git commit -m 增加测试（对于本次修改的）

6、 git push -u origin master（上传）
