CSS伪类、伪元素初步认识

W3C 两者的定义：

  • CSS 伪类用于向某些选择器添加特殊的效果。
  • CSS 伪元素用于将特殊的效果添加到某些选择器。

伪类的种类

伪元素的种类

伪类与伪元素的区别

伪类的效果可以通过添加一个实际的类来达到，而伪元素的效果则需要通过添加一个实际的元素才能达到，这也是为什么他们一个称为伪类，一个称为伪元素的原因。

应用

1、伪类的运用
  a:link {color: #FF0000}  
a:visited {color: #00FF00}  
a:hover {color: #FF00FF}  
a:active {color: #0000FF}
2、::before和::after运用

（1）.实例

.div::before{
    content: "我是before";width:50px;height: 50px;display: block;
    position: absolute;top: -10px;left: -10px;background: gray;
    color: white;border-radius: 50%;
}
.div::after{
    content: "我是after";width:50px;height: 50px;display: block;
    position: absolute;top: -10px;right: -10px;background: gray;
    color: white;border-radius: 50%;
}

（2）清除浮动

 .clearfix::after{
    content: '';
    clear: both;
    display: block;
    height: 0px;
    overflow: hidden;
    zoom: 1;
}

（3）中间固定，两端自适应布局运用

3、first-line和first-letter运用
.div1::first-line{
    font-size: 30px;background: red;color: white;
}
.div1::first-letter{
    font-size: 50px;color: black;
}
4、::selection运用
::selection{
    background-color: yellow;color: white;
}
注：selection只可以设置background-color和color
