Ajax网络请求函数封装

  • Javascript

什么是Ajax

Ajax的全称是 Asynchronous JavaScript And XML （异步Javascript和XML）,它不是一项新的技术，而是由多种技术组合而成

Ajax技术说的是把Javascript、Css、Dom和HTML结合起来的一种新用法。这种结合并不是新的概念–事实上DHTML技术早就把这几样技术结合在一起了。Ajax的独到之处在于它在服务器端使用异步（Asynchronous）处理技术

1.创建XMLHttpRequest对象

XMLHttpRequest对象并不是一个标准。在不同的浏览器中有不同的实现。
IE浏览器语法：

var ajax=new ActiveXObject("Microsoft.XMLHTTP")

其他浏览器语法：

var ajax=new XMLHttpRequest（）；

兼容性写法

if(window.XMLHttpRequest){
    var request = new XMLHttpRequest();
}else{
    var request = new ActiveXObject("Microsoft.XMLHTTP");
}

2.创建一个请求

创建XMLHTTPrequest后，调用open（）方法创建一个请求并向服务器发送.
方法说明：

open(method, url, asynchronous, arg);

Method:指定请求类型一般为get或者post;

url：指定请求地址，可以是绝对地址
asynchronous：同步false异步true默认异步true；
arg：请求参数；

3.发送请求

在创建一个请求后，可以使用send（）方法向服务器发送这个请求；

方法说明：send(content);

该方法有一个参数content，它表示向服务器发送数据，没有参数是默认为null；

避免乱码设置请求头，如下：

request.setRequestHeader("Content-type","application/x-www-form-urlencoded");

4.捕获请求的状态

XMLhttpRequest对象提供了onreadystatechange事件来捕获请求的状态;

request.onreadystatechange=function(){
    if (request.readyState==4&&request.status==200) {
        var response=request.responseText;
        if (successCallback) {
            successCallback(response);
        }
    }else if (request.readyState==4&&!request.status!=200) {
        if (errorCallback) {
            errorCallback(request.statusText);
        }
    }
}

5.判断请求状态和请求结果

request.readyState==4&&request.status==200表示请求数据返回成功和服务端响应成功

6.返回请求结果
使用responseText属性可以获取请求页面的纯文本内容

封装好的请求函数

function requestByGET(url,successCallback,errorCallback){
if(window.XMLHttpRequest){
    var request = new XMLHttpRequest();
}else{
    var request = new ActiveXObject("Microsoft.XMLHTTP");
}
request.open("GET",url,true);
request.send();
request.onreadystatechange = function(){
    if (request.readyState==4&&request.status==200) {
        //如果调用函数的时候，传入了成功的回调函数，我们调用一下这个回调函数，并把服务器返回的数据作为回调函数的参数，供外界使用
        if (successCallback) {
            successCallback(request.responseText);
        }
    }else if (request.readyState==4&&request.status!=200) {
        if (errorCallback) {
            errorCallback(request.statusText);
        }
    }
}
}
function requestByPOST(url,postbody,successCallback,errorCallback){
if(window.XMLHttpRequest){
    var request=new XMLHttpRequest();
}else{
    var request=new ActiveXObject("Microsoft.XMLHTTP");
}

//设置请求对象的请求地址
request.open("POST",url,true);

//设置请求头部的请求格式为键值对方式传输
//multipart/form-data 以文件的形式传输内容
request.setRequestHeader("Content-type","application/x-www-form-urlencoded");

request.send(postbody);
request.onreadystatechange=function(){
    if (request.readyState==4&&request.status==200) {
        var response=request.responseText;
        if (successCallback) {
            successCallback(response);
        }
    }else if (request.readyState==4&&!request.status!=200) {
        if (errorCallback) {
            errorCallback(request.statusText);
        }
    }
}

}
