微信开发基础

  • weChat

序

这篇文章主要将微信开发的基本流程和网站开发的思路，学会在项目实战中学习知识，使自己朝大牛的路上更近一步。

一.申请微信公众账号

完成微信的开发，我们需要申请微信的公众号(服务号/订阅号)。这两者是有区别的，自己申请的时候就可以看到；申请链接不知道的可以自行百度，由于我申请的是个人账号，大部分开发都没有权限，所以只好用微信的测试号来开发，将来部署的时候直接替换就可以。

二.配置基本信息

我们申请页面成功之后就可以看到具体的 测试号信息 和 接口配置信息 ，我们需要配置接口的信息；配置接口信息我们需要一个服务器和域名(可以是服务器的IP）；
我们需要填写的是：
url 你自己的服务器ip （路径下的wx_sample.php文件是在微信接入指南下载的点此即可下载）

Token 值需要和URL下wx_sample.php定义的php值一样

js接口安全域名在开发jssdk的时候可以用到，此时也可以填上

这个页面的OAuth2.0也要修改，要不然在授权时会提示redirect_uri参数错误

三.微信开发常用函数

配置完成后详细的步骤可以查看[微信网页开发]介绍，主要附上几个常用函数

1.网页授权相关函数

//发起get请求函数
function httpGet($url){
    $cURL=curl_init();
    curl_setopt($cURL, CURLOPT_URL, $url);
    curl_setopt($cURL, CURLOPT_RETURNTRANSFER, true);

    $response=curl_exec($cURL);
    curl_close($cURL);
    return $response;
}
//发起post请求函数
function httpPost($url,$postData){
    $cURL=curl_init();
    curl_setopt($cURL, CURLOPT_POST, true);
    curl_setopt($cURL, CURLOPT_URL, $url);
    curl_setopt($cURL, CURLOPT_POSTFIELDS, $postData);
    curl_setopt($cURL, CURLOPT_RETURNTRANSFER, true);
    $response=curl_exec($cURL);
    curl_close($cURL);
    return $response;
}

            return "errcode:".$obj->errcode.",errmsg".$obj->errmsg;
        }else{
            return $obj->access_token;
        }
    }

//生成签名的随机串
function create_nonceStr ($length = 16){
    $chars = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
    $str = "";
    for ($i = 0; $i < $length; $i++) {
      $str .= substr($chars, mt_rand(0, strlen($chars) - 1), 1);
    }
    return $str;    
}
