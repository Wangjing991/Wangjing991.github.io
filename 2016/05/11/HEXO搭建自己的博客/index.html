HEXO搭建自己的博客

  • HEXO 博客

前言

本文是基于Mac 搭建个人博客教程，里面介绍了Node.js、Hexo和Git的安装设置，以及域名的注册

Hexo简介

hexo是一个基于node.js的静态博客程序。hexo使用Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

工具安装

配置Xcode环境

在AppStore下载Xcode安装即可

安装Node

作用：用来生成静态页面的

在Node.js官网下载相应平台的最新版本，一键安装即可

使用以下命令验证是否安装成功
node -v
npm -v

安装Git

作用:把本地的hexo内容提交到github上去

安装Xcode就自带有Git

安装Markdown

作用：简单的标记语言，对文档进行编写

点击Markdown下载安装即可

安装Hexo

Node和Git都安装好后,首先创建一个文件夹,如blog,用户存放hexo的配

置文件,然后进入blog里安装Hexo。

执行如下命令安装hexo：

sudo npm install -g hexo（全局安装）

hexo init （init初始化hexo）

hexo generate或者hexo g （生成静态页面）

hexo server （启动本地服务，进行本地预览，生产本地连接http://localhost:4000）

配置Github

在Github网站操作

1.建立Repository

2.建立与用户名相同的库，【your_user_name.github.io】 固定写法

注：your_user_name必须是用户注册Github的账号名

3.命令：vim _config.yml

与本地建立的_config.yml文件建立关联

4.翻到下面修改下面两条代码

type: git

repo:https://github.com/user_name/user_name.github.io.git

5.命令：npm install hexo-deployer-git –save
