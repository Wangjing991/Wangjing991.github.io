PHP对文件操作

  • php

最近自学了一些php文件操作的常用命令，并自己搭建了一个文件的增删改查系统，前面是php基本语法，后面附上文件操作的几个重要函数

1.fopen:是file open的缩写。 用于打开一个文件。有2个必须要写的参数：文件路径 和 打开方式 fopen返回一个resource类型的数据。

2.filesize函数作用是获取指定文件的大小。 //参数是一个文件名（含路径） //返回值是文件大小，单位是字节。

3.fseek是用于更改文件光标位置的。 //有三个参数，文件，偏移量，参考位置。 //参考位置有3种，SEEK-SET,SEEK-CUR,SEEK-END

4.fgetc函数是读取文件的一个字符，读完之后，文件光标会自动往后移动一位

5.fread函数，读取指定长度的内容。读够了length或者到文件结尾会自动结束。

  1. file 读取文件中的内容，生成一个数组（\n作为分隔）

7.fclose是file close的缩写。它的功能是关闭一个已经打开的文件。参数是一个resource类型的变量。返回值是一个bool，true表示关闭成功，false表示关闭失败。

8.fwrite作用是往文件里面写入指定数据。如果写了参数3，限制了长度，那么我们参数二只会写入指定长度的字符。 //返回值是一个整数，告知你写入了多少个字符。

9.file_put_contents函数是往指定文件里面写入内容，默认打开方式是w，会清空原有内容。 // file_put_contents是fopen，fwrite，fclose的封装。

copy是复制文件，第一个参数是原文件，第二个参数是复制之后新的文件的名字 // $isSuccess = copy(“1.txt”,”2.txt”); // var_dump($isSuccess);
11.rename是给文件改名字，第一个参数是旧名字，第二个参数是新名字。 // $isRenameSuccess = rename(“2.txt”,”3.txt”); // var_dump($isRenameSuccess);

12.unlink作用是 删除指定的文件。如果文件不存在，会报错，如果文件存在，文件会被删除。 unlink是不可以删除文件夹的。 // $isDeleteSuccess = unlink(“3.txt”); // var_dump($isDeleteSuccess);

13.时间设置函数 date_default_timezone_set(“Asia/Shanghai”);

// filectime是文件的最后修改时间。修改包括，修改文件的内容，权限，所有者等等。返回的是一个时间戳。
$ctime = filectime(“1.txt”);

// date的功能是 将时间戳按照指定的格式进行转换。
// 返回值就是转换之后的时间
$time = date(“Y-n-j H:i:s”,$ctime);
echo “$time”;
14.opendir 作用是打开指定文件夹。返回值是一个resource类型的变量。 $dir = opendir(“.”);

15.readdir读取指定目录下的文件名，读完一个之后，会自动后移一位，便于下一次读取下一个文件名。 // $filename = readdir($dir);

16.scandir 获取指定目录下的所有文件名，存放到一个数组中去。返回值是一个数组，数组里面保存了全部的文件名。 //scandir是opendir、readdir、closedir的封装。 // $files = scandir(“.”); // print_r($files);

17.rmdir 删除指定的文件夹，如果文件夹不为空，删除失败。我们需要做的就是先删除文件夹里面的全部文件，再删除这个文件夹

function delFolder($path){

    while(($item=readdir($handle))!==false){
        if($item!="."&&$item!=".."){
            if(is_file($path."/".$item)){
                unlink($path."/".$item);
            }
