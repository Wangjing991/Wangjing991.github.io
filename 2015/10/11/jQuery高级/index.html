jQuery高级

  • jQuery

这篇介绍jQuery高级的运用

1.get()就是把JQ转成原生的JS
        下标表示集合中的第几个元素
    $("#div").get(0).innerHTML
    for(var i=0;i<$('li').get().length;i++) //get()将一组元素转换成一组li集合
    {
        $('li').get(i).style.background='red'
    }
    for(var i=0;i<$('li').length;i++) //jq下的length
    {
        $('li').get(i).style.background='red'
    }
    for(var i=0;i<$('li').length;i++) 
    {
        $('li').[i].style.background='red' //用双括号代替get[]
    }
    2.outerWidth()/outerHeight可以获取显示或者隐藏元素的距离
        offsetWidth获取不到隐藏元素的距离
    3.Text获取文本中的所有内容
        HTML可以获取标签
    4.remove()删除节点后事件消失(点击)
      detach()删除节点后事件存在保留元素的所有操作行为(点击)
      $(function(){
        $('div').click(function(){
        alert(123);
        });
        var oDiv = $('div').detach();
        $('body').append( oDiv );
    });
    5.$(function(){ //等dom加载完就可以执行，性能要好

    })
    $(document).ready(function(){

    })
    6.parents获取元素的所有祖先节点(包括html.body.div.)
        parents('body')参数是起筛选作用的
    例:$('#div1').parent().css({background:"red"})    

     closest():获取最近的指定的祖先节点(祖先节点包括当前元素自身)
     必须含有参数,只能找到一个元素
    例:$('#div1').closest("body").css({background:"red"})

    siblings: 找所有的兄弟节点,参数也是可以筛选的
    $('span').siblings().css('background','red');
    nextAll() : 下面所有的兄弟节点，参数也是筛选功能
    $('span').nextAll().css('background','red');
    prevAll() : 上面所有的兄弟节点
    Until() : 截止
    $('span').nextUntil('h2').css('background','red');

    7.clone()克隆节点;
    可以接收一个参数,作用可以复制之前的操作行为clone(true)
    $('div').click(function(){
        alert(123);
    });
    $('div').clone(true).appendTo( $('span') );
    8.wrap
    wrap() : 包装
    wrapAll() : 整体包装
    wrapInner() : 内部包装
    unwrap() : 删除包装 ( 删除父级 : 不包括body )
    $(function(){
    //$('span').wrapInner('<div>');
    $('span').unwrap();
    });

    9.add(),slice()
    var elem = $('div');
    var elem2 = elem.add('span');
    elem.css('color','red');
    elem2.css('background','yellow');

    $('li').slice(1,4).css('background','red');

    10.serialize()串联数据
       serializeArray()串联成数组

    11.animate():
     第一个参数:{}运动的值和属性
     第二个参数:时间(运动快慢) 默认:400;
     第三个参数:运动形式 只有两种( 默认:swing 满快慢 linear 匀速)
     第四个参数:回调函数
     $(this).animate({width : 300 , height : 300} , 4000 , 'linear',function(){
        alert(123);
    });

    stop()阻止当前的这次运动
    $("div").stop(true) 阻止后续运动加true
    $("div").stop(true,true) 可以立即停止到指定的目标点
     finish()可以立即停止到所有指定的目标点
     delay(2000)延迟

     12.delegate,function(){
    this.style.background = 'red';
    });
    $('ul').delegate('li','click',function(){
        this.style.background = 'red';
        $('ul').undelegate();
    });
    事件添加在ul上
    delegate事件委托可以提高性能的
    13.trigger主动触发
    14.ev.data 数据传递
        ev.type 事件类型
        ev.target 操作对象
    $('#div1').on('click',{name:'hello'},function(ev){
        alert(ev.data.name)  //hello
        alert( ev.target ) //object HTMLDivElement
        alert(ev.type)    //click
    })
